IMPLEMENTATION NOTE

지금 버전은 ‘프론트 프로토타입’입니다.

현재는 브라우저에서 바로 실행되는 정적 사이트이며, 가이드북 기반 키워드/상황 분류로 답변합니다. 실제 운영형 AI로 만들려면 사내망 또는 Teams에 배포하고, 서버에서 문서 검색(RAG)과 AI API를 연결하면 됩니다.

  • 외부 이미지 의존 없이 MISTO 로고를 SVG로 구현
  • 가이드북 주요 항목을 지식베이스로 구조화
  • 답변마다 관련 페이지 번호와 신뢰도 표시
  • 개인정보/예외 케이스는 HR 확인 문구로 방어
