IMPLEMENTATION NOTE

UI는 V1로 고정하고, 내용만 V2로 업데이트했습니다.

이 파일은 서버 없이 열리는 단일 HTML입니다. 실제 GPT API가 연결된 것은 아니지만, 질문 키워드와 프로세스 카드 기반으로 AI처럼 답변하도록 구성했습니다. 운영형으로 확장할 때는 이 UI를 그대로 두고, 답변 데이터만 사내 문서 검색/RAG로 교체하면 됩니다.

  • V1의 상단바 · 히어로 · 챗봇 카드 · 시나리오 카드 구조 유지
  • V2의 CASE 판단 → 처리 프로세스 → 필요 자료 → 주의사항 답변 방식 적용
  • 연차/근태/ERP/급여/증빙/복지/업무지원 주제 확대
  • 주민등록번호·계좌번호·비밀번호 등 민감정보 입력 방어
  • 브라우저 단독 실행, UTF-8, 한글 시스템 폰트 기반
처리되었습니다.
